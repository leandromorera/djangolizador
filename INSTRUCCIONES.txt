1- ejecutar en una terminal de window:
djangoproy.bat nombre_proyecto nombre_aplicacion

2- copiar el archivo generate_crud.py al directorio del proyecto.
ubicarnos en el directorio del proyecto y usar python:

3- editar las lineas de generate_crud.py con el nombre de nuestro proyecto y nuestra aplicacion.

os.environ.setdefault("DJANGO_SETTINGS_MODULE", "myproject11.settings")
# Configurations
PROJECT_NAME = "myproject11"
APP_NAME = "myapp11"

4- Ejecutar dos veces seguidas para que registre la aplicacion. (pequeño detalle)
python generate_crud.py

5- python manage.py startapp myapp12

6- Aplicar la generacion de los templates, editar myproject to the name of the project en las lineas:

# Setup Django environment (adjust `my_project.settings` to your project settings module)
os.environ.setdefault('DJANGO_SETTINGS_MODULE', 'myproject13.settings')
django.setup()
# Define the base directory for the templates
BASE_TEMPLATE_DIR = os.path.join('myapp13', 'templates', 'myapp13')

y aplica:
python generate_templates.py


7-Es muy importante editar el archivo base.html para que nuestros templates puedan cargar la informacion:
en la ruta myproyecto/myapp/templates please edit the file 

base.html

<!DOCTYPE html>
<html>
<head>
    <title>{% block title %}My Django App{% endblock %}</title>
</head>
<body>
    <header>
        <h1>Welcome to My App</h1>
        <p>Debug: This is base.html</p>
    </header>
    <main>
        {% block content %}
        {% endblock %}
    </main>
</body>
</html>
